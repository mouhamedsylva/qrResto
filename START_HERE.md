### 1. Backend
```bash
cd qr-order-api
npm run start:dev
```

### 2. Frontend
```bash
cd qr-order-owner
npm run dev
```

### 3. Accéder à l'application

## 🏗️ Architecture

```
Page (TablesAndQr.tsx)
  ↓ Navigation uniquement
Composants (UI et interactions)
  ↓ Utilisation des hooks
Hooks (Logique métier)
  ↓ Appels aux services
Services (Appels API)
  ↓ Communication
Backend (API REST)
```

### Avantages
- ✅ **Modulaire** : Séparation des responsabilités
- ✅ **Réutilisable** : Composants indépendants
- ✅ **Maintenable** : Code organisé
- ✅ **Testable** : Chaque partie isolée
- ✅ **Scalable** : Facile d'ajouter des features
