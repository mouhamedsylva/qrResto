# 🎨 Guide Utilisateur - Éditeur Visuel Professionnel

## 🚀 Démarrage Rapide

### Accéder à l'Éditeur
1. Ouvrir **Tables & QR**
2. Cliquer sur l'onglet **Personnalisation QR**
3. Sélectionner une table
4. Cliquer sur **"Générer le QR Code"**
5. Cliquer sur **"✏️ Éditeur Visuel"**

---

## 🛠️ Outils Disponibles

### Ajouter des Éléments
| Outil | Icône | Description |
|-------|-------|-------------|
| **Texte** | 📝 | Ajoute un texte éditable |
| **Rectangle** | ▭ | Ajoute un rectangle |
| **Cercle** | ⭕ | Ajoute un cercle |
| **Dupliquer** | 📋 | Duplique l'élément sélectionné |

### Actions Rapides
| Action | Raccourci | Description |
|--------|-----------|-------------|
| **Copier** | Ctrl+C | Copie l'élément sélectionné |
| **Coller** | Ctrl+V | Colle l'élément copié |
| **Annuler** | Ctrl+Z | Annule la dernière action |
| **Refaire** | Ctrl+Y | Refait l'action annulée |
| **Supprimer** | Del | Supprime l'élément sélectionné |

---

## ✏️ Modifier un Élément

### 1. Sélectionner
- Cliquer sur l'élément dans le canvas
- Ou cliquer dans l'onglet **"Calques"**

### 2. Déplacer
- Glisser-déposer avec la souris
- Ou modifier X et Y dans les propriétés

### 3. Redimensionner
- Modifier Largeur et Hauteur dans les propriétés
- Ou utiliser la poignée en bas à droite

### 4. Personnaliser

#### Pour le Texte
- **Double-clic** pour éditer le contenu
- **Taille**: 12px à 96px
- **Style**: Gras, Italique, Souligné
- **Alignement**: Gauche, Centre, Droite
- **Couleur**: Sélecteur de couleur

#### Pour les Formes
- **Couleur de fond**: Sélecteur de couleur
- **Bordure**: Épaisseur (0-20px) et couleur
- **Arrondi**: Pour les rectangles (0-50px)

#### Propriétés Communes
- **Opacité**: 0% à 100%
- **Rotation**: 0° à 360°
- **Position**: X, Y précis
- **Ordre**: Premier plan / Arrière-plan

## 💾 Sauvegarder comme Template

### Étapes
1. Créer votre design complet
2. Cliquer sur **"💾 Sauvegarder comme Template"**
3. Entrer un nom (ex: "Menu Été 2026")
4. Cliquer sur **"💾 Sauvegarder"**
5. ✅ Votre template est ajouté!

### Où le retrouver?
- Retourner à l'onglet **"📋 Templates Prédéfinis"**
- Voir la section **"⭐ Mes Templates Personnalisés"**
- Votre template apparaît avec sa miniature

---

## 📋 Utiliser un Template Personnalisé

### Étapes
1. Aller dans **"Personnalisation QR"**
2. Voir la section **"⭐ Mes Templates Personnalisés"**
3. Cliquer sur le template désiré
4. Aperçu immédiat du design!
5. Sélectionner une table
6. Cliquer sur **"Générer le QR Code"**
7. Télécharger ou Imprimer

---

## 🗑️ Gérer les Templates

### Supprimer un Template
- Cliquer sur le **×** en haut à droite du template
- Confirmer la suppression

### Supprimer Tous les Templates
- Cliquer sur **"🗑️ Tout supprimer"**
- Confirmer l'action

## 🎯 Conseils & Astuces

### Organisation des Calques
1. Utiliser l'onglet **"Calques"** pour voir tous les éléments
2. Les éléments en haut de la liste sont au premier plan
3. Cliquer sur un calque pour le sélectionner rapidement

### Alignement Précis
1. Utiliser les champs X et Y pour un positionnement exact
2. Exemple: X=250, Y=350 pour centrer à 500x700

### Couleurs Cohérentes
1. Noter les codes couleur utilisés (ex: #d94a6a)
2. Réutiliser les mêmes couleurs pour la cohérence
3. Utiliser le sélecteur de couleur pour copier une couleur existante

### Textes Lisibles
1. Taille minimum: 18px pour la lisibilité
2. Contraste: Texte foncé sur fond clair (ou inverse)
3. Police: Utiliser Gras pour les titres

### QR Code Visible
1. Taille minimum: 150x150px
2. Fond blanc autour du QR pour le contraste
3. Ne pas superposer d'éléments sur le QR

## 📱 Exemples de Designs

### Design Minimaliste
```
- Fond blanc
- QR code centré (200x200px)
- Texte en haut: "SCANNEZ POUR COMMANDER" (32px, gras)
- Nom du restaurant en bas (24px)
- Numéro de table (18px)
```

### Design Coloré
```
- Fond dégradé (forme rectangle pleine page)
- QR code sur fond blanc (180x180px)
- Texte avec ombre (opacité 80%)
- Formes décoratives (cercles, rectangles)
- Couleurs vives et contrastées
```

### Design Élégant
```
- Fond beige/crème
- Bordure décorative (rectangle avec bordure)
- QR code dans un cadre
- Texte en police élégante
- Éléments subtils (opacité 50%)
```

## ❓ FAQ

### Q: Combien de templates puis-je sauvegarder?
**R:** Illimité! Ils sont stockés dans votre navigateur.

### Q: Les templates sont-ils partagés entre utilisateurs?
**R:** Non, chaque utilisateur a ses propres templates.

### Q: Puis-je modifier un template sauvegardé?
**R:** Oui! Sélectionnez-le, générez un QR, ouvrez l'éditeur, modifiez, et sauvegardez sous un nouveau nom.

### Q: Que se passe-t-il si je vide le cache du navigateur?
**R:** Les templates seront supprimés. Pensez à les exporter régulièrement.

### Q: Puis-je exporter mes templates?
**R:** Actuellement, ils sont dans localStorage. Une fonction d'export/import sera ajoutée prochainement.

### Q: Le QR code fonctionne-t-il après impression?
**R:** Oui! Assurez-vous qu'il soit assez grand (minimum 150x150px) et bien contrasté.

### Q: Puis-je utiliser mes propres images?
**R:** Cette fonctionnalité sera ajoutée dans une prochaine version.

## 📞 Support

### Problème avec l'éditeur?
1. Rafraîchir la page (F5)
2. Vider le cache du navigateur
3. Vérifier que JavaScript est activé
4. Essayer un autre navigateur (Chrome recommandé)

### Template ne s'affiche pas?
1. Vérifier dans l'onglet "Templates Prédéfinis"
2. Regarder la section "Mes Templates Personnalisés"
3. Si vide, créer un nouveau template

### QR code ne se génère pas?
1. Vérifier la connexion internet
2. Sélectionner une table valide
3. Attendre quelques secondes
4. Réessayer
